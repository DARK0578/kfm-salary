# 工资计算器 需求文档

## 概述

基于钉钉导出的打卡记录 xlsx，一键生成 打卡汇总 + 工资计算 + 其他支出 + 财务汇总 的 HTML 单页工具。

---

## 1. 文件输入

- 支持拖拽或点击选择 **钉钉打卡时间报表** (.xlsx/.xls)
- 使用 SheetJS 解析，自动提取月份和人员数据
- 同一月份重复导入会**覆盖更新**，保留手动新增的人员和配置
- 新月份自动创建历史 Tab

## 2. 打卡记录汇总

### 2.1 数据展示

| 功能 | 说明 |
|------|------|
| 表格列 | 姓名 + 1~31日 + 出勤天数 |
| 表头 | 显示日期号和星期几 |
| 周末列 | 红色背景标记（周六/日） |
| 全局切换 | 📊 出勤天数 / 🕐 打卡时间 两种视图 |

### 2.2 半天判定

```
当日工作时长 = 最后一次打卡 − 第一次打卡
>= 5小时 → 1天（绿色 1）
> 0 且 < 5小时 → 0.5天（黄色 0.5）
= 0 → 缺勤（灰色 ·）
```

- 工作时长通过实际打卡时间自动计算
- 半天日自动标记为 0.5

## 3. 工资计算

### 3.1 工资表结构

| 列 | 类型 | 说明 |
|----|------|------|
| 姓名 | 只读 | — |
| 部门 | 只读 | — |
| 方式 | 下拉 | 月薪 / 日薪 |
| 金额 | 输入框 | 月基本工资 或 日薪 |
| 出勤 | 输入框 | 可手动修改（自动计算值） |
| 应出勤 | 输入框 | 可手动修改（自动计算值） |
| 应发 | 只读 | 自动计算 |
| 月中支出 | 输入框 | 每人当月扣款（如借支、罚款） |
| 实发 | 只读 | 应发 − 月中支出 |
| 备注 | 输入框 | 自由备注 |
| 操作 | 按钮 | 移除该人员 |

### 3.2 计算公式

```
月薪制：应发 = (出勤天数 / 应出勤天数) × 月薪
日薪制：应发 = 出勤天数 × 日薪
实发 = 应发 - 月中支出
```

### 3.3 配置项

- **固定休息日**：默认 2 天
- **应出勤天数** = 当月总天数 − 固定休息日
- 修改休息日自动重算所有人员
- 出勤和应出勤均可手动编辑

## 4. 其他支出

| 功能 | 说明 |
|------|------|
| 列表结构 | 项目名称 + 金额 |
| 增删 | + 添加 / ✕ 删除 |
| 自动合计 | 实时计算总支出 |
| 存储 | 跟随当前月份持久化 |

## 5. 财务汇总

### 5.1 核心卡片（深色突出）

```
┌────────────────────────────┐
│  本月总计                   │
│  ¥19,517.93    ← 超大字号   │
│  实发 + 月中扣 + 其他 + 上月结余 │
├────────────────────────────┤
│  本月实际转入   [20,000.00] │
│  上月结余       [500.00]    │ ← 可手动（可负）│
├────────────────────────────┤
│  本月剩余       +¥482.07    │ ← 自动 = 转入 - 总计
└────────────────────────────┘
```

### 5.2 计算公式

```
本月总计 = 实发工资合计 + 月中扣除合计 + 其他支出合计 + 上月结余
本月剩余 = 本月实际转入 − 本月总计
```

## 6. 人员管理

| 操作 | 说明 |
|------|------|
| + 新增人员 | 弹出输入框填写姓名、部门，出勤默认为 0，手动编辑出勤天数 |
| 移除 | 从当前月移除，其他月不受影响 |
| 手动标记 | 手动新增的人员显示「手动」标签 |
| xlsx 导入合并 | 重新导入时保留手动新增的人员 |

## 7. 历史月份

- 所有月份数据存储在 **浏览器 localStorage**
- 顶部 Tab 栏列出所有历史月份
- 点击 Tab 切换查看
- 支持删除当前月份
- 上月结余自动从上月流转：`当月"上月结余"默认值 = 上月"本月剩余"`
- 打开页面自动恢复最近月份

## 8. 导出

| 格式 | 方式 |
|------|------|
| Excel | 点击「导出 Excel」，生成含 3 个 Sheet 的 xlsx（打卡汇总 / 工资计算 / 支出与汇总） |
| PDF | 点击「打印 / 导出 PDF」，浏览器打印功能，A3 横版 |

## 9. 数据持久化

- 本地存储 key：`kfm_v5`
- 数据结构：
  - `emps`：员工配置（月薪/日薪、金额）
  - `months`：按月份存储所有数据
- 关闭页面后数据不丢失
- 换浏览器/设备需重新导入 xlsx
